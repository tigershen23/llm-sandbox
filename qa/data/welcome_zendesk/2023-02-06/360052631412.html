Recording Events

Best Practices

  • Starting a recording takes a few seconds (about 10-15 seconds). It's best to start recording a little bit earlier than the event's start to ensure you capture everything (i.e. don't start the recording just as you push a speaker to the stage). You can always cut extra footage during the editing process.
  • You can start and stop the recording as many times as you'd like. It will create a new file each time.
  • You cannot record breakout rooms and lounges.
  • If your event is over three hours, the best practice is to start and stop the recordings during intermissions to create new video files in the system. Extremely long videos (3+ hours) are prone to corruption/error and will not generate an SRT file (captions).
  • Double-check to make sure that you've hit "stop record" at the end of your event. This is very important. If you forget to do this the recording might become corrupted/unplayable.
  • If you need to change your event time to enable recordings, please make sure you set the time at least 15 minutes in the future to allow the recording server to spin up.

How To Record

In the Control Room above and to the right of the main stage will be the record button in red

    • The recording button will appear approximately one hour before the start time of your event. Otherwise, it is greyed out.

Where To Find Recordings

Recordings are stored in the Event Dashboard.

    • Go to the left sidebar and toward the bottom is the "Recording" section. Within there, click on the "Download Recording" button to download the video file.
      • Please note that recordings never expire. You will have access to them as long as you have access to Welcome.
