10/18/22 - Chat Updates

Welcome back for another round of fixes and some new enhancements. We hope everyone has been enjoying our new Agenda Builder. We have got your feedback and applied some fixes, along with other bug fixes. We pushed out our new chat updates to help create more fun in the chat channel for your attendees.

Enhancements

Chat Updates

  • Add more emojis to chat tray along with a giphy button
  • Chat Reactions! Users can now use emojis on user chat messages. Simply hover over someones chat message and a pop up will appear with emoji selection.
  • Renamed "GR" chat to "Speakers" chat
  • Clean up chat indicators to include message counts

Bug Fixes

  • Chat exports
  • Fix known cases where duplicating events would not work
  • Fix known cases where “Greyed Out” agenda items would appear after deleting event assets
  • Fix flashing video when switching between tabs with overlay on stage
  • Several bugfixes for agenda builder

Still need help?

If none of the solutions work above and you're still experiencing issues, feel free to contact support@experiencewelcome.com or click Help at the bottom right corner of the screen.
