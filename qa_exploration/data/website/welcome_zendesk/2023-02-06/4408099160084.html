Analytics Guide

Once your event is over, it's time to take a look at your data from those that attended. We have two areas in our analytics tab to help you out.

First is our Event Report Card, this is a one-pager report that provides some top-down actionable insights. The second will be more granular detailed data at a per-user level.

Where to Find Your Data

When you close the doors to your event you will be given an option to receive an email of the Event Report Card PDF. If you choose to not do that you can access your data within your Event Dashboard. 

Please navigate to the analytics section of your Event Dashboard. 

The following data is currently displayed:

  • Event Report Card
  • Attendance
  • Chat
  • Q&A
  • Polling 
  • Breakouts
  • Lounges
  • On Demand
  • 1:1 Calls

All of the data can be downloaded in individual CSV files. A button to do so is provided in the top right-hand corner of each section. Please note that you will need to wait 90 minutes after your event ends for the data to become available.

Multi-track event data is NOT fully yet supported; please reach out to support@experiencewelcome.com for assistance pulling data from these events. 

Event Report Card

This is where you will find top-down data for your event across various metrics. You will also be able to download a PDF file by clicking the "Download PDF" button at the top right.

The Event Report Card is broken up into different sections:

Avg Time Spent at Event in Minutes

Avg Number of Engagements Per Attendee

Conversion - This will show the distinct count across the conversion funnel starting from invited to engage during the event.

    • Invited
    • Completed Registration
    • Attended Event
    • Engaged During Event

Engagement - This encompasses when an attendee performed one or more of the different tracked engagement metrics.

    • Joined Breakout
    • Joined Lounge as a Participant
    • Took Photobooth Picture
    • Clicked Lounge Call to Action
    • Responded to Poll
    • Sent Chat Message
    • Clicked Primary Info Tab
    • Clicked Secondary Info Tab
    • Asked Q&A
    • Voted Q&A
    • Joined Green Room
    • Went on Stage
    • Raised Hands

Engagement Over Time

Please note, that staff, speakers, and Welcome employees are excluded from the data in the Event Report Card.

Times are displayed in the event's time zone. You can find this in the event details tab of your Welcome event dashboard.

Attendance

Complete attendance data can be pulled via this tab. The data will show when the attendee entered the event, exited the event, and the total time spent.

In the chart above you will see columns:

  • First Name
  • Last Name
  • Email
  • Role
  • Company
  • Stage Name
  • Time Spent
  • Time Spent in Seconds

Chat

Your public chat log will be downloadable from here. This is the only chat log that can be downloaded.

Q&A

All Q&A questions will be accessible from here. You will be able to see the users that asked the question and how many upvotes they received. Check out our Q&A Guide if you would like to learn more about the feature. 

Polling

All Poll results will be available here. Check out our Polls Guide if you would like to learn more about the feature.

Breakout

A breakdown of the breakout sessions is housed here. Similar to the attendee data you will see the same user multiple times if they entered and exited it multiple times. Check out our Breakout Room Guide if you would like to learn more about the feature.

Lounge

A breakdown of the lounge sessions is housed here. Similar to the attendee data you will see the same user multiple times if they entered and exited it multiple times. Check out our Lounge Guide if you would like to learn more about the feature.

On Demand

Your post-event On Demand video stats is housed here. This section will show who watched your On Demand content and for how long. Check out our Video On Demand Guide if you would like to learn more about the feature.

1:1 Calls

All 1:1 Call sessions can be found here. Check out our 1:1 Call Guide if you would like to learn more about the feature.
