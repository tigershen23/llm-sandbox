5/13/22

Welcome back for another round of updates, fixes, and features!

Features

  • BETA:
    • Email Customization: Added the ability for producers to customize the copy of Welcome’s Emails (manual emails, automated emails, and reminders). Please ping support@experiencewelcome.com if you are interested in this feature. Also, please note that there are a few quirks that we are working on with this feature (detailed below).
      • Registrant's first names are sometimes not filled in with the user's name. Please remove this section when customizing the email until a fix is in.
      • Emojis in the email body text should be rendered in-line.
      • The email sender name defaults to the name of the event in the Email Customization Dashboard.
      • Hyperlinking is not supported at this time.
      • At this moment the email formatting cannot be changed.
    • QA Written Replies: This allows you to provide written responses to questions in the QA tab. Please ping support@experiencewelcome.com if you are interested in this feature.
      • At this time this feature is only available for desktop users.
    • Enhanced Event Toolkit: Adds some default assets to new events (overlays and backgrounds in your brand's colors). Additionally, the Stage Kit has been updated to include some new brand-colored assets. Finally, this feature also adds nametag overlays in your brand's colors. Please ping support@experiencewelcome.com if you are interested in this feature. 
      • We are working on performance improvements for this feature at this time.
    • Translation Service Support: Producers can add a translation service in the Event Details in the Event Dashboard. You will need to utilize streaming through the service that you choose for this feature to work. Please ping support@experiencewelcome.com if you are interested in this feature.

Enhancements

  • Added an embed a form for producers to send referrals from the event dashboard.
  • Users can now easily share on-demand recordings or a registration page through social media.
  • Multi-track events are now supported on mobile. Attendees will be able to easily swap between stages via a drop-down at the top of their screen.
  • Changes made to Info Tabs will now update without refresh. Additionally, any changes to any other stage details will now update live.
  • Changed the view of the organization dashboard to separate past events from upcoming events.
  • Added a warning when producers set their event time to less than 15 mins in the future that recording, streaming, and mobile will not work.
  • Removed the counts for people online that are broken down by user type (Attendees, Speakers, Staff) but kept the aggregate count for all people online.
  • Updated the formatting of the “Send Invites”, “Registration Confirmation”, 24hr and 1hr Reminder emails.

Fixes

  • Fixed a casing issue for SSO where Welcome will still redirect to SSO provider even if a user provides improper casing in their email domain (e.g. support@ExperienceWELCOME.com instead of support@experiencewelcome.com).
