Help, my Attendees/Speakers can only see a black screen and no audio!

One of the most common issues you will hear from users is "Has it started yet? I only see a black screen and no audio". This is most likely due to either a VPN, some sort of network security like a firewall, and/or a security extension installed in their browser. 

Of the three ways this could happen, a VPN is the most common. The simplest solution to this is to ask if they are able to disable the VPN. This may not be possible in all cases, so let us break down the different ways this can happen with possible solutions.

 

VPN 

  • Turn off VPN if possible.
  • Try a different browser or an incognito/private window.
  • Use a different device; generally, we recommend a personal device as these are less likely to have security installed on the device.
  • View the event on mobile. Attendees will still be able to utilize the chat, Q&A, and polls on mobile. However, they will not be able to join lounges or breakout.
  • If you are recording the event, offer a way to view the event recording.

Firewall

  • Move to a personal network such as using a mobile phone as a wifi hotspot. Sometimes a Guest network will work.
  • Offer your IT team our Networking — troubleshooting & tuning for optimal performance guide.
  • View the event on mobile using a mobile data plan. Mobile allows for all of Welcome's functionality except for joining breakouts or lounges.
  • If you are recording the event, offer a way to view the event recording.

Browser Security

  • Use an incognito/private window of the browser. This will generally disable installed extensions that could be blocking the feed.
  • Manually disable the security extension that was either installed by IT security or even pop-up blockers.

Still need help?

If none of the solutions work above and you're still experiencing issues, feel free to contact support@experiencewelcome.com or click Help at the bottom right corner of the screen.
